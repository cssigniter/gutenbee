GutenBee enhances the Gutenberg editor with more blocks!

== Description ==
GutenBee provides new blocks for the Gutenberg editor.

**Blocks:**

* Spacer
* Divider

With more on the way...

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/gutenbee` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the "Plugins" screen in WordPress
3. New Gutenberg blocks will appear!
